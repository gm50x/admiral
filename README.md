# Super Simple CRUD API
Yet another Node-Postgres-Express-Mocha-API

# Getting Started
* start your database (we are using Postgres)
* start the project (on development mode)

        npm install
        npm run dev

* [OR] start the project

        npm install
        npm start

* [OR] run the tests

        npm install
        npm test
